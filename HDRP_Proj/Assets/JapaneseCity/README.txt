

--------------------
Japanese City
--------------------


Description:
--------------------
Japanese City contains the assets to make up the downtown area of Japan.

Content
--------------------
- 2 Demo scenes(Day/Overview)
- 182 FBX
- 310 Prefabs
- 220 Textures (Type:Albedo ,Metalic ,Normal and Emissive Map/Size:512- 4096)
- Buildings have 2 LOD's

Triangle Count
----------------------
Prop asset: 10 - 4000
Building Prefabs: 1000 - 60000

Info
--------------------------
*Buildings have no interior.
*This package contains 3D art only, no rigs, scripts or music.
*All screenshots and videos are captured on PC build using HDPR(2020.3.42f1).

Supports URP
Supports HDRP (a .unitypackage file included)
